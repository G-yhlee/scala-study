# 타입, 표현식, 값

```
# 타입
- 컴파일 타임에 존재
- Int, Float 타입 모두 메모리에서 32-bit 로 표현된다

# 표현식
- 컴파일타임에 존재
- 값으로 변환된다
- 타입을 가진다

# 값
- 런 타임에 존재
- 모든 값은 객체다
- 값은 메모리에 존재
- 값은 타입을 가지지않는다
- ( 예시: 2 라는 표현식은 특정한 비트로 평가되어서 메모리에 저장 )


# 2 / 0 ( Int, type erasure , 런타임에러, 컴파일타임 존재 )
- 위의 표현식은 , Int 타입을 가진다
- 컴파일 타임에 존재한다
- 런타임에 에러가 난다
- 스칼라는 컴파일타임에 모든 에러를 잡아주지 못하기 때문에, 이러한 에러를 핸들링하는 코드 디자인이 매우 중요하다
- 컴파일 타임에는 타입을 가지지만 , 런타임에서는 타입에 대한 정보가 남지 않는데, 이러한 과정을 [ type erasure ] 라고 표현한다
```
